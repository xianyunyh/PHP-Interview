## 生成自己的 Gitbook

```bash
$ npm install gitbook-cli -g
$ git clone https://github.com/xianyunyh/PHP-Interview
$ cd PHP-Interview
$ gitbook serve # 本地预览
$ gitbook build # 生成静态的 HTML
```
